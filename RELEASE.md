# 💾 SaveVault v1.0.0 - Initial Release

Welcome to the first official release of **SaveVault**! 
SaveVault is a modern desktop utility designed to effortlessly back up and restore your video game save files using your personal Google Drive storage. Whether you're moving to a new PC or syncing between laptops, SaveVault remembers exactly where your files and registry keys belong.

### ✨ Key Features in this Release
* **Seamless Google Drive Backup:** Safely upload save folders, individual save files, and even Windows Registry keys straight to your Google Drive.
* **One-Click Restore:** Downloads and automatically places your files and registry keys back into their exact original locations.
* **Auto-Watch (Background Sync):** Automatically monitors your game directories and triggers backups when it detects that you've saved your game.
* **Modern Interface:** A sleek, user-friendly Dark Mode GUI built with CustomTkinter.
* **Portable & Standalone:** Packaged as a single `.exe` file! No need to install Python or mess with command lines.

### 🚀 How to Install & Use
1. Download the **`main.exe`** file attached below (you can rename it to **`SaveVault.exe`**).
2. Double-click the file to launch it (no installation required!).
3. Click **"Connect Google Drive"**. Your browser will open asking you to authorize SaveVault with your Google Account.
4. Start adding your games and backing up!

### ⚠️ Important Note regarding Google Login
Because SaveVault modifies files on your Google Drive directly, Google may show a **"Google hasn't verified this app"** warning during your first login. This is completely normal for newly created Google Cloud projects! 
* To bypass it: Click **Advanced** -> **Go to SaveVault (unsafe)** and grant the permissions.
